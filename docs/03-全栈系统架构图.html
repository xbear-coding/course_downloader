Course_Downloader — 全栈系统架构 v1.0 | FastAPI + Vue3 + SQLite + Playwright

🎯 业务上下文

目标

  • 统一管理 6 个平台的课程/视频/文章离线下载
  • 自动分类归档到本地目录
  • 视频自动转码 MP4（H.265）+ ASR 文字转写
  • 增量检测 — 新课自动识别、更新课程提示重下

用户场景

  • 熊子熠 — 腾讯会议家教课程 + 小鹅通付费课 + B站教程
  • 抖音/头条 — 收藏的教学视频和文章离线保存
  • 小红书 — 教程笔记图文离线
6 平台 视频下载 ASR 转写 增量检测 错误重试 Web UI

🚀 部署信息

运行环境

  • Windows 11 Pro / Python 3.12+
  • Node.js 18+（前端编译需要）
  • Chrome（Playwright 会自动安装 Chromium）
  • ffmpeg 7.1（已验证 H.265 支持）

启动方式

  • make install — 安装全部依赖
  • make dev — 启动开发服务器（后端 8000 + 前端 5173）
  • make test — 运行测试
关键性能基线（实测）：SQLite 单条写入 0.02ms / Playwright 实例 687MB / ASR ~1s 每段 / 网络下载 ~133KB/s
